# 左侧菜单

顶部工具栏主要实现全局功能，比如保存、预览、发布等，暂时不支持源码导出，后续会支持。

左侧工具栏主要实现组件的拖拽，组件大纲（点击树形大纲可选中画布中的组件）、页面列表、组件 JSON 源码、接口管理、变量管理、页面成员。
